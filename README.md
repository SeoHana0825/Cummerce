# CH 2 커머스 과제

객체 간의 역할과 책임에 대한 고민을 통한 객체 지향 설계 상품관리 프로그램을 익혀보자

## 프로젝트 소개

* Product 객체 생성을 통해 상품 정보 세팅 : new 활용
* 여러 Product 추가 : List 활용
* product 탐색 : 반복문 for
* 상품 관리, 사용자 입력 처리 클래스 나누기
* start() 함수 메서드를 만들어 반복문 로직 관리
* 객체 지향 설계를 통한 클래스 세분화 및 관
* 캡슐화 적용 : getter, setter 메서드로 관리
